Snake Game

For the snake game I decided to go with an embedded JavaScript script. This game, although fun to play, is one of the easier games to make. All I needed was a canvas to draw the items, functions to track positions, logic to handle collisions, and event listeners for keys.

Within the script I collect the canvas and initialize variables to contain all necessary elements. The snake, at it's simplest, is an array of coordinates within the canvas grid. As the snake eats the apple we cause the snake to grow where the head is always at the front of the array. We also access a p tag labeled as score to update with each apple consumption. As the snake moves in the grid we draw based on the position and or where the users chose to move the head. The game will reset if the snake collides with its self.

Overall, only four functions were needed to implement this game. This being a smaller game I felt embedded was the easier way to go.
